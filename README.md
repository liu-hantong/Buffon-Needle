# 蒲丰投针模拟
## 介绍  
通过`Unity`自带物理引擎实现一个近真实的投针环境，模拟蒲丰投针实验  
关于实验原理解释，参看[Buffon's Needle Problem-Wikipedia](https://en.wikipedia.org/wiki/Buffon's_needle)
